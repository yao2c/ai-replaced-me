# AI替代我了吗 - 病毒式测试网站

## Context
做一个有趣的测试网站，用户通过10道题测试自己被AI替代的程度，最终生成可截图分享的结果卡片。目标：在社交媒体上病毒传播。

## 技术方案
单文件 `index.html`（HTML + CSS + JS 内联），零依赖，可直接打开或部署到 Vercel。

## 项目目录
```
~/ai-replaced-me/
├── index.html      # 全部代码
└── vercel.json     # 可选，Vercel部署配置
```

## UI 流程（4个屏幕）

1. **首页** — 标题 "AI替代我了吗？"，副标题 "测测你还能苟多久"，开始按钮
2. **答题** — 每次一题，进度条，4个选项卡片
3. **计算中** — 假装AI扫描动画（打字机效果，3秒）
4. **结果卡片** — 可截图分享的卡片 + 再测一次按钮

## 10道题目设计

每题4个选项，分值1-4（1=难替代，4=易替代），毒舌风格：

1. 你的日常工作主要是？（安抚甲方情绪/做PPT汇报/写代码/发呆开会）
2. 你最核心的职场技能？（读懂领导微表情/把简单事情说复杂/Excel数据分析/画图写文案）
3. 同事需要你通常因为？（修电脑打印机/知道哪个外卖好吃/做脏活/精神支柱）
4. AI工具用得怎样？（靠AI干活/偶尔用ChatGPT/觉得AI是智障/AI是什么能吃吗）
5. 公司裁员你觉得自己...（第一批/中间偏危险/应该还好吧/裁我公司不敢）
6. 加班原因？（领导没走/活太多/摸鱼补白天的活/从不加班）
7. 工作成果AI能几分钟复制？（30秒甚至更好/七八成/能做但拉胯/做梦做不出）
8. 最近被夸因为？（方案好/人nice/执行力强/上次被夸是什么时候）
9. 最不可替代的是？（人脉关系/审美创意/技术能力/工位风水好）
10. 面对AI崛起你的态度？（拥抱变化学prompt/焦虑但刷手机/无所谓/转行卖烤红薯）

## 结果卡片设计
- 暗色背景 (#1a1a2e)，赛博朋克风
- 中心：圆环进度条动画显示替代百分比
- 等级标签 + 毒舌描述
- 底部："长按截图分享给同事 让他们也焦虑一下"
- 尺寸适配手机截图 (375x667)

## 视觉风格
- 深色主题，霓虹色点缀
- 移动端优先，max-width 420px 居中
- CSS transition 做屏幕切换动画
- Canvas API 绘制可截图卡片（零依赖）

## 实现顺序
1. HTML 骨架（4个屏幕）
2. CSS 深色主题 + 动画 + 响应式
3. JS 题目数据 + 答题逻辑 + 计分
4. JS 加载动画（打字机效果）
5. JS 结果卡片渲染（HTML + Canvas截图版）
6. OG meta 标签（微信/微博分享）
7. 手机测试 + 部署

## 验证方式
- 浏览器直接打开 index.html 测试全流程
- 手机端截图测试卡片效果
- `vercel deploy` 部署预览
